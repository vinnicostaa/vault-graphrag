# Superlógica — ERP condominial líder BR

## Posicionamento
- Maior ERP condominial do Brasil, cloud, foco em **administradoras** (B2B), não síndico direto.
- Também atua em imobiliárias e assinaturas (SaaS multi-vertical).

## Features centrais
- **Financeiro**: boletos (Pix, bancos), inadimplência, fluxo de caixa, conciliação bancária automática, 160+ relatórios.
- **Cobrança**: CRM de cobrança, notificações SMS/e-mail, integração PBAX (2a via por telefone reduz ligações -50%).
- **Comunicação**: avisos, circulares, documentos, portal morador (2a via boleto, reservas, dados).
- **Assembleia virtual**: integrada via Zoom (somente plano Enterprise).
- **Outros Enterprise**: Porteiro Online, reserva áreas comuns, gestão de compras, malotes, veículos.
- **App morador**: "Gruvi" (próprio; críticas sobre adoção — exige % alta de instalação p/ funcionar).

## API & Integrações
- **API v2**: RESTful, JSON, OAuth2, HTTPS. Construída em parceria com **Sensedia**.
- Sandbox disponível para desenvolvedores.
- Webhooks disponíveis.
- App Token gerado dentro do ERP (Todos os usuários → API → Aplicações → Novo App Token).
- Documentação em `superlogica.com/developers` + `api@superlogica.com`.
- Marketplace de integrações: parceiros (ex.: Organize Meu Condomínio integra via API).

## Pontos fracos (Reclame Aqui)
- Cobrança indevida, dificuldade cancelar.
- App Gruvi exige adesão massiva dos moradores p/ funcionar.
- PDFs misturam documentos e comunicados no sistema.
- Revogação de acesso sem aviso.
- Taxa de resolução RA: **88,8%** (bom), tempo médio 5d7h.

## Relevância para Master Síndico
- É o **benchmark de ERP** — Master Síndico NÃO compete nesse eixo (foco em governança).
- API pública robusta = potencial parceiro (integração bidirecional: puxar dados financeiros do ERP Superlógica para camada de governança).
- Superlógica publica "Índice Superlógica" (taxa condomínio BR 2025 +6,8%) — fonte de benchmark de mercado.
