# Group Software — Condomínio21 / Group Condomínios

## Posicionamento
- Grupo brasileiro com 2 produtos principais:
  - **Condomínio21 (Cond21)**: ERP tradicional instalado (desktop + web complementar), foco administradoras médio/grande porte.
  - **Group Condomínios**: versão 100% online, sucessora/alternativa.
  - **Condomínio21 One**: variação para autogestão (síndico direto).

## Features Cond21
- Contas a receber/pagar, previsão orçamentária detalhada.
- **Contabilidade com 15+ modelos de balancete** (diferencial forte em contabilidade condominial).
- Conta-pool p/ múltiplos condomínios, fluxo de caixa diário/semanal/mensal (projetado vs realizado).
- Comunicação administrador ↔ morador.
- Gestão de cobrança, inadimplência.
- App Condomínio21 Medições (leitura consumo).

## Assembleia híbrida
- Group Software publica conteúdo técnico sobre assembleia híbrida; produto suporta presencial + remoto conforme Lei 14.309/2022.

## Críticas (Reclame Aqui)
- Sistema considerado "não completo" por alguns usuários ("péssima experiência").
- Curva de aprendizado alta (legado desktop).

## Relevância para Master Síndico
- Cond21 é padrão em administradoras tradicionais de médio porte (alternativa B2B à Superlógica).
- Não tem reputação de prestador, não tem LMS, não tem marketplace estruturado.
- Pode ser integrado via dados no futuro (não tem API pública tão documentada quanto Superlógica).
