# [Padrões]{.dp1-h-1} [de]{.dp-h-article} [Projeto]{.dp1-h-2} {#padrões-de-projeto .dp1-h .dp-abs .dp-c .dp-h1}

::: mob-image
[![Padrões de
Projeto](../images/patterns/content/index/patterns-i140f4.png?id=ccf20fa726f8a7349011e4dff2979773){width="256"
height="192"
srcset="/images/patterns/content/index/patterns-i1-2x.png?id=9522cfd9ae822396f2b4c3f5932e5992 2x,/images/patterns/content/index/patterns-i1-3x.png 3x"}](design-patterns/what-is-pattern.html "Introdução aos padrões de design")
:::

**Padrões de projeto** (design patterns) são soluções típicas para
problemas comuns em projeto de software. Cada padrão é como uma planta
de construção que você pode customizar para resolver um problema de
projeto particular em seu código.

::: {.dp1-b .dp-abs .dp-c .dp-b}

projeto?](design-patterns/what-is-pattern.html){.btn .btn-primary}
