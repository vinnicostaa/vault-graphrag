# Породжувальні патерни проектування {#породжувальні-патерни-проектування .title}

Список породжувальних патернів проектування, які відповідають за зручне
та безпечне створення нових об\'єктів або навіть цілих сімейств
об\'єктів.

::: {.d-flex .flex-wrap .justify-content-between}
[[ [![Фабричний
метод](../../images/patterns/cards/factory-method-minid7f6.png?id=72619e9527893374b98a5913779ac167){srcset="/images/patterns/cards/factory-method-mini-2x.png?id=fa9d4a8d61a67cc3822e52b9daf69dad 2x,/images/patterns/cards/factory-method-mini-3x.png 3x"}]{.pattern-image}
[Фабричний метод]{.pattern-name .with-aka} [Factory
Method]{.pattern-aka} ]{.pattern-card-top} [
]{.pattern-card-bottom}](factory-method.html){.pattern-card-extended
.factory-method}

Визначає загальний інтерфейс для створення об\'єктів у суперкласі,
дозволяючи підкласам змінювати тип створюваних об\'єктів.

[[ [![Абстрактна
фабрика](../../images/patterns/cards/abstract-factory-minid1c5.png?id=4c3927c446313a38ce77dfee38111e27){srcset="/images/patterns/cards/abstract-factory-mini-2x.png?id=22236aaa65ff52cbde1c713216d52c1f 2x,/images/patterns/cards/abstract-factory-mini-3x.png 3x"}]{.pattern-image}
[Абстрактна фабрика]{.pattern-name .with-aka} [Abstract
Factory]{.pattern-aka} ]{.pattern-card-top} [
]{.pattern-card-bottom}](abstract-factory.html){.pattern-card-extended
.abstract-factory}

Дає змогу створювати сімейства пов'язаних об'єктів, не прив'язуючись до
конкретних класів створюваних об'єктів.

[[
[![Будівельник](../../images/patterns/cards/builder-mini6d24.png?id=19b95fd05e6469679752c0554b116815){srcset="/images/patterns/cards/builder-mini-2x.png?id=de6d0938678b86903a1426dddfdd13bf 2x,/images/patterns/cards/builder-mini-3x.png 3x"}]{.pattern-image}
[Будівельник]{.pattern-name .with-aka} [Builder]{.pattern-aka}
]{.pattern-card-top} [
]{.pattern-card-bottom}](builder.html){.pattern-card-extended .builder}

Дає змогу створювати складні об\'єкти крок за кроком. Будівельник дає
можливість використовувати один і той самий код будівництва для
отримання різних відображень об\'єктів.

[[
[![Прототип](../../images/patterns/cards/prototype-mini6540.png?id=bc3046bb39ff36574c08d49839fd1c8e){srcset="/images/patterns/cards/prototype-mini-2x.png?id=b871f789a736e7efbb1cd082d2de6398 2x,/images/patterns/cards/prototype-mini-3x.png 3x"}]{.pattern-image}
[Прототип]{.pattern-name .with-aka} [Prototype]{.pattern-aka}
]{.pattern-card-top} [
]{.pattern-card-bottom}](prototype.html){.pattern-card-extended
.prototype}

Дає змогу копіювати об\'єкти, не вдаючись у подробиці їхньої реалізації.

[[
[![Одинак](../../images/patterns/cards/singleton-minif559.png?id=914e1565dfdf15f240e766163bd303ec){srcset="/images/patterns/cards/singleton-mini-2x.png?id=4a7348083d7719d02709a5c9ff878b9f 2x,/images/patterns/cards/singleton-mini-3x.png 3x"}]{.pattern-image}
[Одинак]{.pattern-name .with-aka} [Singleton]{.pattern-aka}
]{.pattern-card-top} [
]{.pattern-card-bottom}](singleton.html){.pattern-card-extended
.singleton}

Гарантує, що клас має лише один екземпляр, та надає глобальну точку
доступу до нього.
:::
