# Рефакторинг {#рефакторинг .ref1-h .ref-abs .ref-c .ref-h1}

::: mob-image
[![Вступ в
Рефакторинг](../images/refactoring/content/index/refactoring-i166e6.png?id=4ba85db5ddb3da0cd829bc7c0d2bf407){width="256"
height="192"
srcset="/images/refactoring/content/index/refactoring-i1-2x.png?id=dc9209e9ccee993f51c33697c716aa49 2x,/images/refactoring/content/index/refactoring-i1-3x.png 3x"}](refactoring/what-is-refactoring.html "Вступ в Рефакторинг")
:::

Рефакторинг --- це контрольований процес покращення коду, без написання
нової функціональності. Результат рефакторингу --- це чистий код і
простий дизайн.

.btn-primary}
:::

::: mob-image
[![Чистий
код](../images/refactoring/content/index/refactoring-i3179f.png?id=e393546f4f2d0046fdbf98b75c6c35ea){width="256"

передбачуваність процесу роз- робки та підвищує якість кінцевого
продукту.

::: mob-image
