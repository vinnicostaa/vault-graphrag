# [Patrones]{.dp1-h-1} [de]{.dp-h-article} [diseño]{.dp1-h-2} {#patrones-de-diseño .dp1-h .dp-abs .dp-c .dp-h1}

::: mob-image
[![Patrones de
diseño](../images/patterns/content/index/patterns-i140f4.png?id=ccf20fa726f8a7349011e4dff2979773){width="256"
height="192"
srcset="/images/patterns/content/index/patterns-i1-2x.png?id=9522cfd9ae822396f2b4c3f5932e5992 2x,/images/patterns/content/index/patterns-i1-3x.png 3x"}](design-patterns/what-is-pattern.html "Intro to design patterns")
:::

Los **patrones de diseño** (design patterns) son soluciones habituales a
problemas comunes en el diseño de software. Cada patrón es como un plano
que se puede personalizar para resolver un problema de diseño particular
de tu código.

::: {.dp1-b .dp-abs .dp-c .dp-b}
[ ¿Qué es un patrón de
diseño?](design-patterns/what-is-pattern.html){.btn .btn-primary}
:::
