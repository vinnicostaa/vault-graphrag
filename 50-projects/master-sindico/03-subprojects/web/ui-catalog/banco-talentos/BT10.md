# BT10 — Consentimento LGPD

## Finalidade
Coletar consentimento explícito e versionado do morador sobre uso dos dados para recrutamento e seleção. Obrigatório antes de transitar de `draft` para `submitted`.

## Fonte canônica
- `vault/90-archive/master-sindico-pdfs-cliente-originais/ESTRUTURA DE CADASTRO DE CURRÍCULO.md` §TELA 10
- `vault/50-projects/master-sindico/00-product/sub-produtos/07-banco-de-talentos.md` §8 BT10

## Texto canônico (literal)
> "Declaro que as informações fornecidas são verdadeiras e autorizo o uso dos meus dados exclusivamente para fins de recrutamento e seleção, conforme a LGPD.
>
> Estou ciente de que a Master Síndico não garante contratação e atua apenas como plataforma de conexão entre profissionais e empresas."

## Persona e ABAC
- **Persona**: Morador Pagante (`resident`)
- **Plan tier**: `base` pagante
- **ABAC action**: `content.talent.lgpd.consent.self`

## Componentes
- `LegalTextBlock` (mostra `terms_version` vigente)
- `MandatoryCheckbox` ("Concordo com os termos")
- `PrimaryButton` ("Finalizar cadastro") — desabilitado até checkbox marcado
- `MetaInfo` (data/hora/IP no momento do aceite)

## Estados
- **Inicial**: checkbox desmarcado, botão desabilitado.
- **Submitting**: spinner.
- **Success**: redireciona [[BT11]] confirmação.
- **Error**: toast retry.
- **Termo atualizado posteriormente**: morador é notificado e precisa **reconsentir** para manter `published`.

## Integração com backend
| Ação | Endpoint (inferido) | Método |
|---|---|---|
| Carregar termo vigente | `/api/v1/talents/lgpd/template` | GET |
| Confirmar consentimento | `/api/v1/talents/curriculum/me/lgpd-consent` | POST `{ terms_version, accepted: true }` |

## Regras de negócio críticas
- **INV-BT-04**: `lgpd_consent_version IS NOT NULL AND lgpd_consented_at IS NOT NULL` quando `status ∈ {submitted, published}`.
- Sem aceite → `Curriculum` permanece `draft` (não pode submeter).
- Quando termo atualizado → notificação + reconsent obrigatório para manter `published`.
- Versão do termo (`terms_version`) é parte do registro probatório.
- Logs em tabela central `lgpd_logs` (cross-módulo, retenção 5 anos).
