# BT05 — Dados Pessoais

## Finalidade
Coletar dados básicos do candidato. Empresa NÃO vê contato direto desses dados — apenas via formulário de convite (Connect Me Empresa→Morador BT).

## Fonte canônica
- `vault/90-archive/master-sindico-pdfs-cliente-originais/ESTRUTURA DE CADASTRO DE CURRÍCULO.md` §TELA 05
- `vault/50-projects/master-sindico/00-product/sub-produtos/07-banco-de-talentos.md` §6.1

## Mensagem institucional canônica
> "Vamos aos seus dados básicos. Esses dados não são usados como critério de exclusão."

## Persona e ABAC
- **Persona**: Morador Pagante (`resident`)
- **Plan tier**: `base` pagante
- **ABAC action**: `content.talent.personal_data.update.self`

## Campos obrigatórios (PDF cliente)
- Nome completo
- Telefone
- E-mail
- CEP (utilizado **apenas para identificar bairro**)
- Idade
- Estado civil
- Possui filhos? ( ) Sim ( ) Não
- Carteira de habilitação (CNH): ( ) Não ( ) A ( ) B ( ) AB
- Cursos NR: NR-10 / NR-33 / NR-35 / Outros: _____

## Componentes
- `PersonalDataForm`
- `CEPLookup` (preenche bairro automaticamente — só bairro, não endereço completo)
- `CNHRadioGroup` (4 valores)
- `NRMultiSelect` (4 valores: NR-10, NR-33, NR-35, outros)
- `LegalNote` (anti-discriminação)
- `PrimaryButton` ("Continuar")

## Estados
- **Inicial**: vazio (ou pré-preenchido se vindo do registro Master Síndico).
- **Validating**: inline (CPF, email, CEP).
- **Submitting**: spinner.
- **Success**: redireciona [[BT06]].
- **Error**: toast retry.

## Integração com backend
| Ação | Endpoint (inferido) | Método |
|---|---|---|
| Lookup CEP (apenas bairro) | `/api/v1/utils/cep/{cep}/neighborhood` | GET |
| Salvar dados | `/api/v1/talents/curriculum/me/personal` | PATCH |

## Regras de negócio críticas
- **CEP usado apenas para identificar bairro** (privacidade).
- **Empresa NÃO vê telefone/email** — só via formulário de convite Connect Me especial Empresa→Morador BT.
- Dados sensíveis: estado civil, filhos — **NÃO podem ser usados como critério de exclusão** (compromisso anti-discriminação canônico).
- LGPD: tratamento como dado pessoal sensível; retenção 5 anos pós-archive.
- CNH e NR são marcadores operacionais úteis para vagas condominiais.
