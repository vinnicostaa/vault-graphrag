# C11 — Dossiê de Governança Exportável

## Finalidade
Gerar PDF estruturado consolidando todas as evidências documentais da gestão: declarações, assinaturas, conflitos, mapa de risco, auditoria, contratos, termos, LGPD e indicadores. Produto final usado em assembleia, processo judicial, auditoria, seguro.

- `backend/.kiro/specs/master-sindico/requirements/cross-domain.md` Req 40 + Req 42

## Persona e ABAC
- **Persona primária**: Síndico principal
- **Plan tier mínimo**: `plus` (export é benefício pago)
- **ABAC action**: `compliance.dossier.export`
- **Restrições**: gera evento próprio em [[C5]] auditoria; cada export é registrado.

## Conteúdo compilado (canônico)
- Declarações anuais ([[C2]] — atual + histórico do mandato)
- Assinaturas dos usuários ([[C3]])
- Matriz de conflito ([[C4]])
- Mapa de risco ([[C7]])
- Trilha de auditoria — **resumo** (não o log inteiro; ver [[C5]])
- Contratações ([[C8]])
- Termos assinados (Connect Me + outros)
- LGPD — **resumo** ([[C9]])
- Indicadores ([[C10]] excluído por privacidade — confirmar política)

## Formato
**PDF estruturado** (single file ou ZIP com anexos referenciados — backlog).

## Uso destinado (canônico)
- Assembleia
- Processo judicial
- Auditoria externa (administradora, conselho)
- Seguro (cotação/sinistro)

## Estados
- **Inicial**: configuração + botão.
- **Generating**: spinner + texto "Compilando evidências (10-60s)..."
- **Ready**: card de download com TTL 24h + botão "Gerar nova versão".
- **Failed**: toast "Falha — tentar novamente"; preserva config.
- **Pendências detectadas (gatilho)**: bloqueio com mensagem canônica:
  > "Para prosseguir, finalize as pendências de compliance."
  
  + lista de itens travando (link direto para [[C2]]/[[C3]]/[[C7]]/[[C8]]).

## Regras de negócio críticas — **GATILHO 4 (canônico)**
- **Compliance ativo é OBRIGATÓRIO** para exportar dossiê (gatilho 4 do §"Regras de Gatilho").
- Outros 3 gatilhos: encerrar mandato (gatilho 1), gerar relatório final (gatilho 2), marcar negócio fechado (gatilho 3 — opcional).
- Cada exportação:
  - Gera linha em [[C5]] auditoria
  - Registra evento `compliance.dossier.exported`
  - Inclui hash SHA-256 do PDF para verificação posterior
- Dossiê gerado é **imutável** (URL pública verificável — backlog M4+).
- Geração é assíncrona (job queue), pode levar 10-60s para gestões grandes.
- Signed URL TTL 24h; após expiração, regerar.
