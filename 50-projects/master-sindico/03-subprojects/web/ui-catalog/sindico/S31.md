# S31 — Auditoria (C5 — trilha imutável)

## Finalidade
Trilha de auditoria visível e imutável de todas as ações relevantes do condomínio. Exibe lista estruturada (usuário, perfil, ação realizada, campo alterado, versão anterior, versão atual, data, hora) com filtros. Base probatória append-only retida por 5 anos (LGPD).

## Persona e ABAC
- **Persona primária**: Síndico principal (`syndic`, role_type=`principal`) + Admin MS (transversal)
- **Plan tier mínimo**: `trial`
- **ABAC action**: `audit_log.read`
- **Restrições**:
  - **Acesso restrito**: somente admin MS + síndico principal podem acessar (F1:111 + sub-produto §5).
  - Auxiliary, conselho, administradora **não veem** a trilha (privacidade probatória).
  - Append-only: nenhuma operação de UPDATE/DELETE permitida via UI ou API.

## Fluxo da tela
1. Header "Trilha de Auditoria" + selo "Imutável" + indicador retenção (5 anos LGPD).
2. Filtros:
   - Usuário (autocomplete)
   - Data (range)
   - Tipo de ação (multi-select: criação, atualização, validação, publicação, alteração cadastral, login/logout, mudança de permissão, exclusão lógica/soft delete)
   - Perfil (principal | auxiliary | administradora | conselho | empresa síndica)
3. Busca textual (campo alterado).
4. Tabela paginada (colunas: data/hora · usuário · perfil · ação · campo · valor anterior · valor atual · IP — opcional).
5. Click em linha → drawer com dados completos (incluindo metadados técnicos).
6. Botão "Exportar trilha" (CSV/PDF — feature compliance C11).

## Regras de negócio críticas
- **Append-only enforced em DB-level** (sub-produto §3 + F2:111-142): query pattern nunca permite UPDATE/DELETE em `audit_log_entries`.
- Eventos rastreados (mínimo): criação de atividades, validação de registros, publicação de comunicados, alterações cadastrais, atualização do plano diretor, login/logout, criação/modificação de pauta de assembleia, votações, mudanças de permissões, exclusões soft (F1:99-104 + Req 37).
- **Retenção 5 anos** (LGPD). Após 1 ano, archive em MinIO/S3 Glacier (F1:107).
- Acesso restrito a principal + admin (privacidade).
- IP opcional na tabela; sempre presente no drawer de detalhe.
- Exportação dispara log próprio (auditar a auditoria).
