# M5 — Documentos da Unidade (Mobile)

## Finalidade

Lista documentos anexados à unidade do morador: IPTU, apólice de seguro, comprovante de endereço, contratos. Download + abrir + compartilhar.

## Persona e ABAC

- Morador, `unit.docs.self.read`.

## Fluxo mobile

1. Entrada via M4 ou menu.
2. Lista `GET /institutional/units/{id}/documents`.
3. Tap → abre via `open_filex` (PDF) ou `photo_view`.
4. Download local para offline.

## Widget Flutter principal

`DocumentsListPage` com `ListTile` por doc + ações (download, share, abrir).

## Platform-specific notes

- iOS: share sheet nativa após download.
- Android: `ACTION_VIEW` intent.
- **Permissions**: storage write (Android < 10); iOS usa Documents dir.

## Estados

- Loading / Loaded / Downloading(%) / Error.

## Offline behavior

- Docs baixados ficam em `ApplicationDocumentsDirectory` persistente.
- Metadados cached Hive 7d.

## Push notification trigger

- `unit.docs.{id}.added` quando síndico adiciona novo.

## Gaps/ressalvas

- Upload de doc pelo morador: M3 (via Connect Me proof anexo).
