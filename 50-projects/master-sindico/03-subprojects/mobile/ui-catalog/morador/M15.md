# M15 — Perfil do Morador (Mobile)

## Finalidade

Tab bottom nav "Perfil". Centraliza dados pessoais, memberships, preferências notificação, sessão, conta, logout.

## Persona e ABAC

- Morador autenticado.
- `user.self.read` / `user.self.update`.

## Fluxo mobile

1. Tab Perfil → página com 5 seções: Conta / Vínculos / Notificações / Privacidade / Sobre.
2. Conta: nome, email, CPF (masked), foto.
3. Vínculos: lista memberships → link M13.
4. Notificações: 6 toggles (ver [[../../push-notifications]]).
5. Privacidade: LGPD (export dados, delete conta).
6. Sobre: versão app, termos, licenças, logout.

## Platform-specific notes

- iOS: estilo Cupertino opcional para sections (manter Material por DS).
- Android: confirmation dialog nativo para logout.
- **Biometria**: delete conta exige biometria (M2+).

## Estados

- Loading user / Loaded / Updating (ao trocar toggle) / Error.

## Offline behavior

- User cached Hive session.
- Toggle offline → enfileira em sync queue (M2+).

## Push notification trigger

Nenhum direto; gerencia preferências de outros.

## Gaps/ressalvas

- Upload foto perfil: M2 (presign + crop).
- Export LGPD (ZIP com dados): M3 via link email.
