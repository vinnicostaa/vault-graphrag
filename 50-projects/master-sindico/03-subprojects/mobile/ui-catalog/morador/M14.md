# M14 — Histórico de Avaliações (Mobile)

## Finalidade

Lista read-only das avaliações que o próprio morador já submeteu. Somente visualização dos valores enviados + data + período.

## Persona e ABAC

- Morador, `assessment.self.history`.

## Fluxo mobile

1. Acesso via botão "Meu histórico" em M12 (após submit).
2. `GET /reputation/assessment/history/me` retorna lista.
3. Tap item → detalhe read-only com os 3 critérios + comentário.
