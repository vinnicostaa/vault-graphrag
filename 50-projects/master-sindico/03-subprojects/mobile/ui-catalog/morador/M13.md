# M13 — Meu Vínculo (Mobile)

## Finalidade

Exibe detalhes do vínculo morador-unidade: condomínio, bloco/apto, status (proprietário/inquilino/dependente), fração ideal, data início.

## Persona e ABAC

- Morador, `membership.self.read`.

## Fluxo mobile

1. Acesso via menu Perfil ou home footer.
2. `GET /institutional/memberships/me/{id}` com detalhes completos.
3. Ações: "Encerrar vínculo" (abre modal confirmação + motivo) → `POST /memberships/me/{id}/end`.

## Widget Flutter principal

```dart
class MeuVinculoPage extends StatelessWidget {
  Widget build(BuildContext context) {
    return Scaffold(

}
```

## Platform-specific notes

- Cards com data tables nativos Material 3.
- Haptic warning ao tocar "Encerrar vínculo".

## Estados

- Loading / Loaded / Ending / Ended (redireciona empty home) / Error.

## Offline behavior

- Membership detail cached Hive (reuso memberships list).
- **Encerrar = online only**.

Nenhum.

## Gaps/ressalvas

- Fração ideal é read-only; mudança = síndico via web.
