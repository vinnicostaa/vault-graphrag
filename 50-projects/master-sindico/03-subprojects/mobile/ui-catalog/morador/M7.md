# M7 — Timeline Detail (Mobile)

## Finalidade

Detalhe full-screen de uma entrada da Linha do Tempo. Renderização de markdown + anexos + metadata + share.

## Persona e ABAC

- **Persona**: Morador.
- **ABAC action**: `timeline.read`.

## Fluxo mobile

1. Navegação via tap em item de M6 ou deep-link push.
2. Carrega `GET /timeline/{id}` se não cached; senão renderiza cache imediato.
3. Body: chip categoria + título + autor + data + markdown + anexos.
4. Ações barra inferior: Compartilhar (share sheet nativa) + Imprimir PDF (M3) + Salvar offline (icon).
5. `POST /timeline/{id}/read` auto após 2s na viewport.

## Widget Flutter principal

```dart
class TimelineDetailPage extends StatelessWidget {
  final String entryId;
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(actions: [_ShareButton(entryId)]),
      body: BlocBuilder<TimelineDetailCubit, TimelineDetailState>(
        builder: (_, s) => s.maybeWhen(
          loaded: (entry) => _Body(entry),
          loading: () => const _Skeleton(),
          orElse: () => const _Error(),
        ),
      ),
    );
  }
}
```

- Widget markdown: `flutter_markdown` package + custom builders para imagens/links.
- Attachments: lista com ícones por tipo; tap abre `open_filex` (PDF) ou `photo_view` (img).

## Platform-specific notes

- **iOS**: swipe-back habilitado.
- **Android**: back physical = pop.
- **Both**: `SelectableText` para corpo do markdown permitir copiar trechos.

## Estados

- **Loading**: skeleton header + body.
- **Loaded**: conteúdo completo.
- **Error 404**: "Entrada não encontrada. Pode ter sido removida."
- **Offline cached**: renderiza cache + banner.

## Offline behavior

- Cache Hive `timeline_detail_{id}` 7d.
- Imagens `cached_network_image` persistentes.
- PDF anexo: se baixado 1x, fica em `ApplicationDocumentsDirectory`; botão "baixar" visível offline.

## Push notification trigger

Nenhum trigger próprio; acesso por deep-link push `timeline.{id}.new`.

## Componentes

- `MarkdownBody(content)` customizado
- `AttachmentList(items)`
- `ShareSheet` nativa
- `ReadMarker` (invisible widget que triggers read-state)

## Gaps/ressalvas

- Imprimir PDF exige servidor gerar ou `printing` package client-side; M3.
- Embed vídeo (Mux) em timeline detail: M3+.
