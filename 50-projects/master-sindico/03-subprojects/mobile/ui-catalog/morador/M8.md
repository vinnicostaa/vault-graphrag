# M8 — Plano Diretor (Mobile, View)

## Finalidade

Leitura do Plano Diretor do condomínio. Lista de capítulos + breadcrumb + markdown. **Read-only no mobile** — edição = web (síndico).

## Persona e ABAC

- **Persona**: Morador (any plan).
- **ABAC action**: `plano_diretor.read`.

## Fluxo mobile

1. Card "Plano Diretor" em M1 → navega.
2. Lista capítulos `GET /institutional/plano-diretor/{cond}`.
3. Tap capítulo → M9 detalhe.

```dart
class PlanoDiretorPage extends StatelessWidget {
```

- Lista `ListView.separated`; touch targets 56dp.
- `SafeArea` top + bottom.
