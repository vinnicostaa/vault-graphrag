# M9 — Plano Diretor Capítulo (Mobile)

## Finalidade

Leitura full-screen de um capítulo do plano diretor. Markdown + anchor links entre capítulos + barra de progresso.

## Persona e ABAC

- Morador, `plano_diretor.read`.

## Fluxo mobile

1. Chegada via M8.
2. `GET /plano-diretor/{cond}/chapters/{slug}` retorna markdown.
3. Navegação cap. anterior/próximo via SwipeGesture (`PageView.builder`).

## Widget Flutter principal

```dart
PageView.builder(
  itemCount: chapters.length,
  onPageChanged: (i) => cubit.setIndex(i),
  itemBuilder: (_, i) => SingleChildScrollView(
    padding: EdgeInsets.all(16),
    child: MarkdownBody(data: chapters[i].body),
  ),
);
```

## Platform-specific notes

- Swipe horizontal cap-nav; scroll vertical body.
- iOS: haptic light ao trocar capítulo.
- Android: back pop para M8.

## Estados

- Loading / Loaded / Error / Offline.

## Offline behavior

- Cache Hive `plano_diretor_chapter_{slug}` 24h+.

Nenhum direto.
