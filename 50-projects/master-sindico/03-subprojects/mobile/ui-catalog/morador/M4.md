# M4 — Detalhes da Unidade (Mobile)

## Finalidade

Página com informações estruturais da unidade do morador: bloco, apto, metragem, fração ideal, garagens, dependências cadastradas, histórico de moradores (se síndico liberar).

## Persona e ABAC

- Morador, `unit.self.read`.

## Fluxo mobile

1. Entrada via M13 ("Ver unidade").
2. `GET /institutional/units/{unit_id}` read-only.

## Widget Flutter principal

`UnidadeDetailPage` com `ListView` de `DataRow` cards.

## Platform-specific notes

- Copy-on-tap para campos como código da unidade.

## Estados

- Loading / Loaded / Error / Offline (cache Hive 24h).

## Offline behavior

- Cache Hive `unit_{id}` 24h.

## Push notification trigger

Nenhum.

## Gaps/ressalvas

- Edição de metadados (ex: nome do dependente) = síndico via web.
