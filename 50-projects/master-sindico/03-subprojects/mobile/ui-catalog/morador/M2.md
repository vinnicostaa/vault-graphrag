# M2 — Trocar Condomínio (Mobile)

## Finalidade

Bottom sheet que lista todos os memberships ativos do morador e permite trocar o condomínio default da sessão. Equivalente mobile de [[../../../web/ui-catalog/morador/M2]] (web é modal).

## Persona e ABAC

- **Persona**: Morador com > 1 vínculo ativo.
- **ABAC action**: `membership.default.update`.

## Fluxo mobile

1. Tap no ícone "trocar" do `HeaderCondominio` (M1) → abre `showModalBottomSheet`.
2. Sheet lista memberships (nome condomínio, bloco/apto, badge "atual").
3. Tap em item diferente → `PATCH memberships/me/default { membership_id }` → fecha sheet → M1 recarrega.
4. Botão "Adicionar outro condomínio" → navega para `ONB-M2`.

## Widget Flutter principal

```dart
showModalBottomSheet(
  context: context,
  isScrollControlled: true,
  builder: (_) => BlocProvider.value(
    value: context.read<HomeBloc>(),
    child: const _TrocarCondominioSheet(),
  ),
);
```

- Sheet: `DraggableScrollableSheet(initialChildSize: 0.6, maxChildSize: 0.9)`.
- Lista: `ListView.separated` com `RadioListTile`.

## Platform-specific notes

- **iOS**: bottom sheet com handle bar + rounded top 16dp.
- **Android**: respeitar `MediaQuery.viewInsets` (se teclado aberto ao buscar lista).
- **Both**: swipe-down-to-dismiss nativo.

## Estados

- **Loading**: list skeleton 3 tiles.
- **Success**: lista renderizada + badge "atual".
- **Switching**: tile desabilita + spinner no item clicado.
- **Error**: toast inline "Não foi possível trocar" + botão retry.

## Offline behavior

- Lista de memberships vem de cache Hive (TTL 24h) → renderiza offline.
- **Trocar offline = não permitido** (operação de estado servidor). Banner "Conecte-se para trocar condomínio" + botão desabilitado.

## Push notification trigger

Nenhum direto.

## Integração com backend

| Ação | Endpoint | Método |
|---|---|---|
| Lista memberships | `/api/v1/institutional/memberships/me` | GET |
| Set default | `/api/v1/institutional/memberships/me/default` | PATCH |

## Componentes

- `MembershipTile` (radio + texto bloco/apto + badge)
- `AddMembershipButton` (link para onboarding)

## Gaps/ressalvas

- Busca/filtro só aparece se > 5 memberships; senão lista pura.
- Confirmação antes de trocar? Assumido **não** (troca instantânea é reversível).
