# MOC — Síndico Mobile (subset M1)

Mobile síndico é **subset** do web. M1 inclui **apenas** telas de leitura + entrada. Operações complexas (convocar assembleia, gestão docs, orçamento) = web.

## Telas M1 mobile (subset)

- [[S1]] — Entrada da Governança (primeiro acesso)
- [[S2]] — Seletor de Condomínios
- [[S6]] — Central de Governança (6 cards read-only)
- [[S7]] — Timeline read (síndico view)
- [[S8]] — Compliance view (score + alertas)

## Telas web-only (S9-S31)

Não presentes no mobile em M1. Lista completa em [[../../../web/ui-catalog/sindico/_moc]].

## M2 mobile (Sprint 11)

- Swipe actions "validar pendência"
- Connect Me inbox
- Voice input anotações

## M3 mobile

- Modo apresentação (assembleia)
- Procuração
- Assinatura digital
